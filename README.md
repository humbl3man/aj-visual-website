# AJ Visual Creative Website

This is the official website for AJ Visual Creative, a portfolio website built with SvelteKit and Sanity.

## Features

- **SvelteKit:** A modern, fast, and flexible framework for building web applications.
- **Sanity.io:** A headless CMS for managing content, including projects, landing pages, and site settings.
- **TypeScript:** For type-safe code and better developer experience.
- **Component-Based Architecture:** A modular and reusable component structure.
- **Dynamic Routing:** For project detail pages.
- **Custom UI Components:** Including a carousel, aspect ratio, buttons, and spinners.
- **Sanity Image Component:** A dedicated component for optimizing and displaying images from Sanity.
- **SvelteKit Remote Functions:** Utilizes SvelteKit's experimental remote functions for server-side logic.
